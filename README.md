## About The Project

This is an example 'Bookstore' project.

Its purpose:
* start an initial nestjs project
* create entity "Books", enable CRUD 
* connect to Postgres database


### Built With

* [NestJs](https://nestjs.com/)

## Getting Started

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

* nestjs
  ```sh
  npm install -g @nestjs/cli
  ```
